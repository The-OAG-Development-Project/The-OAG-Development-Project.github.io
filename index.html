OWASP Application Gateway (OAG)HTTP Proxy with benefits.

OWASP Application Gateway is an HTTP proxy that handles OpenIdConnect (and other) authentication, session management, and allows you to control/unify the content of the security tokens (e.g. JWT) used by your applications.
